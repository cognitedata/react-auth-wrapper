# MSAL Angular v1 Samples

The samples below use [`@azure/msal-angular@1`](https://github.com/AzureAD/microsoft-authentication-library-for-js/tree/msal-angular-v1/lib/msal-angular), which uses the implicit flow. For updated samples that use [`@azure/msal-angular@2`](https://github.com/AzureAD/microsoft-authentication-library-for-js/tree/dev/lib/msal-angular) and the auth code flow, please see the `msal-angular-v2-samples` folder [here](https://github.com/AzureAD/microsoft-authentication-library-for-js/tree/dev/samples/msal-angular-v2-samples).
